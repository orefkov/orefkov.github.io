Подробное описание

Проверка, являются ли два типа совместимыми строковыми типами.

Параметры шаблона
K1- первый проверяемый тип.
K2- второй проверяемый тип.

Оба типа должны быть строковыми типами и совпадать по размеру. То есть char и char8_t всегда совместимы, wchar_t тождественен в Linux char32_t, а в Windows - char16_t. Это для возможности смешивать строковые выражения совместимых типов.
