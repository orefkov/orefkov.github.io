template<typename K>
struct simjson::StreamedJsonParser< K >

Парсер текста в JsonValue. Позволяет парсить JSON порциями текста. Например, данные приходят пакетами из сети, скармливаем их в processChunk, пока не получим результат

K- тип символов.
